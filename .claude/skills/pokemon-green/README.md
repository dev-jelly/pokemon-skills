# Pokemon Green - Text RPG Skill

1세대 포켓몬 그린 버전을 텍스트 기반 RPG로 완전 재현한 Claude Code 스킬입니다.

## 기능

- 151마리 포켓몬 (이상해씨 ~ 뮤)
- 165개 기술
- 15타입 상성 (1세대 버그 포함)
- 관동 지방 완전 구현
- 8개 체육관 + 사천왕 + 챔피언
- BGM 자동 재생 (macOS afplay)
- ASCII 아트 스프라이트
- 울음소리 재생

## 사용법

```
포켓몬
새 게임
이어하기
```
