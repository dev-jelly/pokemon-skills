```
  .-*##%#+-:#%#+=+**+*+-        상태: 정상
```


## ASCII 아트 명령어

| 명령어 | 설명 |
|--------|------|
| 도감 [포켓몬] | 도감 정보 + ASCII 아트 |
| [포켓몬] 상세 | 파티 포켓몬 상세 + ASCII 아트 |

## ASCII 아트 데이터 파일

**전체 아트**: `data/sprites/pokemon-ascii.json`
- 151마리 포켓몬 상세 ASCII 아트
- 높이: 19-69줄, 너비: 최대 101자
- 전투 시작, 도감 조회 시 사용

**축소 아트**: `data/sprites/pokemon-ascii-mini.json`
- 151마리 포켓몬 픽셀 스타일 ASCII 아트
- 높이: 13줄 (고정), 너비: 28자
- 문자: `.`, `:`, `-`, `=`, `+`, `*`, `#`, `%`, `@` 사용
- 전투 진행 중, 파티 확인 시 사용

```json
{
  "001": ["line1", "line2", ...],
  "025": ["피카츄 ASCII 아트..."],
  ...
}
```

## ASCII 아트 사용 규칙 (필수 준수)

| 상황 | 사용할 파일 | 설명 |
|------|------------|------|
| **전투 시작** (야생/트레이너) | `pokemon-ascii.json` | 상대 포켓몬 첫 등장 시 전체 아트 1회 표시 |
| **전투 진행 중** (매 턴) | `pokemon-ascii-mini.json` | 축소 아트로 간결하게 표시 |
| **도감 확인** | `pokemon-ascii.json` | 상세 정보와 함께 전체 아트 표시 |
| **파티 상태 확인** | `pokemon-ascii-mini.json` | 선두 포켓몬 축소 아트 |
| **포켓몬 교체** | `pokemon-ascii-mini.json` | 교체 화면에서 축소 아트 |

## ASCII 아트 사용법

**전체 아트 표시 (전투 시작/도감)**:
```python
full_sprites = json.load("data/sprites/pokemon-ascii.json")
art = full_sprites["025"]  # 피카츄
for line in art:
    print(line)
```

**축소 아트 표시 (전투 진행 중)**:
```python
mini_sprites = json.load("data/sprites/pokemon-ascii-mini.json")
mini_art = mini_sprites["025"]  # 피카츄 미니
for line in mini_art:
    print(line)
```
