1세대 포켓몬 그린 버전을 터미널에서 텍스트 RPG로 플레이하는 Claude Code 스킬입니다.

---

## 왜 이걸 만들었나요?

### 향수 (Nostalgia)

1996년, 게임보이 화면에서 처음 만났던 포켓몬. 그 시절의 설렘을 터미널에서 다시 느껴보고 싶었습니다.

### 실험 (Experiment)

이 프로젝트의 진짜 목적은 **"LLM 스킬을 통해 비결정적(non-deterministic) 게임플레이가 가능한가?"**에 대한 실험입니다.

- 기존 게임은 **결정론적**입니다. 같은 입력에 같은 출력.
- 하지만 LLM이 게임마스터 역할을 한다면?
- 매번 조금씩 다른 대화, 다른 뉘앙스, 다른 경험이 가능해집니다.

Claude Code의 스킬 시스템을 통해 **"AI가 진행하는 인터랙티브 픽션"**의 가능성을 탐구합니다.

## 시스템 요구사항

### 필수
- **Claude Code** (Anthropic CLI)
- macOS, Linux, 또는 Windows

### 오디오 (macOS 전용)

> **중요**: BGM, 효과음, 울음소리는 **macOS에서만** 재생됩니다.

이 스킬은 macOS의 `afplay` 명령어를 사용합니다. 다른 OS에서는:
- 게임 자체는 정상 작동
- 음악/효과음 재생만 불가

향후 Linux/Windows 지원을 위한 PR을 환영합니다!

## 설치 및 사용법

### 1. 저장소 클론

```bash
git clone https://github.com/your-username/pukiman.git
cd pukiman
```

### 2. Claude Code에서 실행

```bash
claude
```


또는

- **한국어**: "포켓몬", "새 게임", "전투" 등

## 미래에 대한 상상 (Philosophy)

### "이거 토큰 낭비 아닌가요?"

맞습니다. 솔직히 말해서, 지금 이 프로젝트는 **엄청난 토큰 낭비**입니다.

ASCII 아트 하나 그리는 데 수백 토큰, 전투 한 번에 수천 토큰.
게임보이가 8KB RAM으로 돌리던 걸 우리는 수십억 파라미터 모델로 돌리고 있습니다.

**하지만 이건 미래를 위한 상상입니다.**

LLM 추론이 지금보다 100배, 1000배 빨라지고 저렴해진다면?
- 실시간 게임 진행이 가능해집니다
- 무한한 NPC 대화가 자연스러워집니다
- 플레이어의 모든 행동에 즉각 반응할 수 있습니다

이 프로젝트는 그 미래를 미리 경험해보는 **프로토타입**입니다.
오늘의 "낭비"가 내일의 "당연함"이 될 때를 상상하며 만들었습니다.

### LLM이 게임마스터가 된다면?

이 프로젝트는 단순한 레트로 게임 재현이 아닙니다.

**"AI가 게임을 진행한다"**는 것은 무엇을 의미할까요?

1. **비결정적 경험**: 같은 선택이라도 매번 조금씩 다른 반응
2. **자연어 인터페이스**: 버튼이 아닌 대화로 진행
3. **창발적 스토리텔링**: 정해진 스크립트를 넘어선 즉흥적 대화
4. **무한한 확장성**: 스킬 데이터만 있으면 어떤 게임이든 가능

### 앞으로의 가능성

- **D&D 스타일 TRPG**: AI 던전마스터
- **인터랙티브 소설**: 플레이어 선택에 반응하는 이야기
- **교육용 시뮬레이션**: 역사, 과학 등 체험 학습
- **프로토타이핑**: 게임 아이디어의 빠른 검증

이 프로젝트가 **LLM과 인터랙티브 엔터테인먼트의 미래**에 대한 작은 실험이 되길 바랍니다.
