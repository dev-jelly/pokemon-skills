# Pokemon Green - Text RPG for Claude Code

1세대 포켓몬 그린 버전을 텍스트 기반 RPG로 완전 재현한 Claude Code 스킬입니다.

## 게임 특징

- **151마리 포켓몬**: 이상해씨부터 뮤까지 완전 수록
- **165개 기술**: 모든 1세대 기술 구현
- **15타입 상성**: 1세대 고유 버그 포함 (에스퍼 vs 고스트 0배 등)
- **관동 지방**: 태초마을부터 포켓몬 리그까지
- **8개 체육관**: 웅, 이슬, 마티스, 민화, 독수, 초련, 강연, 비주기
- **세이브/로드**: 10개 슬롯 + 자동저장
- **교환 시스템**: NPC 교환 + 연결의끈 (교환 진화 대체)
- **BGM 자동재생**: 맵 이동/전투 시 배경음악 자동 재생 (macOS)
- **ASCII 아트**: 151마리 포켓몬 ASCII 스프라이트

## 설치 방법

이 저장소를 클론한 후, `.claude/skills/pokemon-green` 폴더를 Claude Code 스킬 디렉토리에 복사하세요.

```bash
git clone https://github.com/your-username/pukiman.git
cp -r pukiman/.claude/skills/pokemon-green ~/.claude/skills/
```

## 전투 시스템

### 데미지 계산 (1세대 공식)
```
기본 데미지 = ((2 × Level ÷ 5 + 2) × Power × A ÷ D ÷ 50 + 2)
최종 데미지 = 기본 데미지 × STAB × 타입상성 × 급소 × 난수
```

### 상태이상

| 상태 | 효과 |
|------|------|
| 독 | 매 턴 HP 1/16 감소 |
| 화상 | 매 턴 HP 1/16, 공격력 1/2 |
| 마비 | 속도 1/4, 25% 행동불가 |
| 잠듦 | 1~3턴 행동불가 |
| 얼음 | 행동불가, 10% 해동 |
| 혼란 | 50% 자해 |

### 1세대 버그 재현

- 에스퍼 vs 고스트: 0배 (원래 2배)
- 포커스 에너지: 급소율 1/4 감소 (원래 증가)
- 독 vs 벌레: 2배 (원래 1배)
